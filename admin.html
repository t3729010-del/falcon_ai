Provider Requirements

🧬 IdentityProvider (IP-Adapter-FaceID)

  • RAM: ~4GB free (SD1.5) or ~8GB (SDXL)
  • Disk: ~5GB for SD1.5 + FaceID weights
  • GPU: Recommended (CPU fallback ~5-10 min)
  • Install: pip install diffusers torch transformers accelerate safetensors insightface opencv-python-headless onnxruntime
  • Preserves: Face structure, eyes, nose, face shape, hairstyle

🖥 LocalProvider (SD2.1)

  • RAM: ~3GB free
  • Disk: ~5GB for model
  • GPU: Optional (CPU fallback ~3-5 min)
  • Install: pip install diffusers torch transformers accelerate safetensors pillow

🎨 ComfyUIProvider

  • ComfyUI: Running on localhost:8188
  • Python: pip install websocket-client
  • Model: Any SD model installed
  • VRAM: Depends on model (2-8GB)

☁ HuggingFaceProvider

  • Free tier: api-inference.huggingface.co
  • Rate limit: ~30 req/min
  • Needs: HF_API_KEY in .env
  • Models: SD2.1, SD1.5 (free)

🌐 BrowserFallbackProvider

  • Zero server cost
  • Runs entirely in browser
  • Canvas-based cartoonization
  • No internet required
  • Always available
